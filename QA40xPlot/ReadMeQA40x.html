Registers

The QA40x has 5 registers used for command/control .

These are:

Writable Register Description Details
0 Read/write test register Whatever is written to this register is also readable as verification.
5 Attenuation Value Range:Value = 0:0, 6:1, 12:2, 18:3, 24:4, 30:5, 36:6, 42:7
6 Generator Output Max Range:Value = 18:3, 8:2, -2:1, -12:0
8 Start or Stop streaming Start:5 Stop:0
9 Sample Rate Rate:Value = 48000:0, 96000:1, 192000:2, 384000:3
13 Read calibration data Value = 16 starts calibration data read from register 25
16 Line Frequency? This reads 60 here
17 USB DC Voltage Read the dc voltage in mv
18 USB DC Current The supply current requirement in mA
19 ?? 4626
21 ?? 1048
22 Temperature In 10*DegreesC
25 Calibration data Read the bytes of calibration data after sending 0x10 to reg 13
