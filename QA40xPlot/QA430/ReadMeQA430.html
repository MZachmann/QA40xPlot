Registers

The QA430 has 5 registers used for command/control .

These are:

Register Description
0 Read/write test register
5 Control bits for relays
8 Negative rail value
9 Positive rail value
10 Enable supply

Where: we have

  • Register 0 is a test register. Writing a value to it and reading it back should return the same value.
  • Register 5 are bit fields for each relay:
    • 0,1 = PosNeg connection: 49.9=b10, Short=b11, Open=b0x
    • 2,3,4,5 = NegInputs: Analyzer+499=b1111, Gnd+4.99=b1110, Open=b110x, Analyzer+4.99K=b10xx, Gnd+499=b0xxx
    • 6,7 = Pos Input: Analyzer=b00, Gnd = b01, Analyzer+100K=b1x
    • 8 = feedback resistor: 4.99K | shorted
    • 9,10,11 = load resistance: 0 = none, 9 = 2K, 10 = 604, 11 = 470 (note doc says 330 sometimes)
    • 12 = current sense: split | no split
    • 13 = use power from: fixed rails | adjustable rails
    • 14 = PSRR LowRail: 0=gnd analyzer | 1=connected to analyzer
    • 15 = PSRR HighRail: 0=gnd analyzer | 1=connected to analyzer
  • Register 8 - negative rail using V = -(byte)(counts/256 * 2.44 * 6)
  • Register 9 - positive rail using V = (byte)(counts/256 * 2.44 * 6)
  • Register 10 - bit 0 enable supply: off | on
  • Register 17 - high side(+) current sense uA
  • Register 18 - low side(-) current sense uA
  • Register 19 - offset voltage uV
  • Register 20 - USB voltage mV

Power and Rails

You can manually set each rail voltage while using adjustable rails.

Fixed rails keeps the voltage at +-15V with >100mA. Adjustable rails can go from 1 to 14.5 volts at lower current (30-50mA) but supports current sensing.

Also rails can be driven by the analyzer.

Default is the rails are fixed and not driven by the analyzer.
